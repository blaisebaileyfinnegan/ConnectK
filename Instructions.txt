Instructions:
------------

1)  Open the workspace file, Connect.dsw using MS Visual C++ 5.0 (other versions may work)

2)  Change the name of the output executable file to your team name by editing the text in the menu Project->Settings->Link->Output file name (you CANNOT leave it as Connect.exe)

3)  Execute the program by choosing, Build->Execute <your program name>.exe

4)  You must provide a competitive AI by modifying the ConnectK.cpp and ConnectK.h files provided with the GUI (Graphical User Interface) source code.  See these files for explainations of the interface between the GUI and the ConnectK class (defined in ConnectK.h)
